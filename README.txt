# Menu Creator
is a personal project for creating, editing a viewing menus.
Stack: NodeJS, ReactJS, MySql.

prerequisite:
-------------
NodeJS installed.
MySql server installed on port 3306, 
uid: "root"
psw: "root"


How to run
-----------
bring dependencies mpdules by npm "install"
run "npm start"
and surf to localhost:8080
